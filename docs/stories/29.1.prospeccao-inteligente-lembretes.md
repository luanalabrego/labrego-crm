## Story

Como **gestor comercial**, quero configurar qual é a etapa de início de prospecção no funil e receber lembretes por email quando a lista de prospects estiver acabando, para manter o fluxo de vendas ativo.

---

## Acceptance Criteria

### AC1: Configurar etapa de início de prospecção
- **Given** a página de configuração de etapas do funil
- **When** o usuário acessa as configurações
- **Then** pode marcar uma etapa como "Etapa de Prospecção"
- **And** a configuração é salva no Firestore

### AC2: Monitoramento da lista de prospecção
- **Given** uma etapa marcada como prospecção
- **When** o número de contatos nela atinge thresholds (300, 200, 100, 0)
- **Then** o sistema registra o alerta

### AC3: Envio de lembretes por email
- **Given** um threshold atingido
- **When** o sistema detecta
- **Then** envia email para os usuários do funil informando o status
- **And** a mensagem inclui quantos contatos restam e orientação para adicionar mais
